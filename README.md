# 📦 Inventory Management System

A full-stack inventory management application built with React, Node.js, Express, and MySQL that enables users to manage inventory items efficiently through a clean and intuitive interface.

## 🚀 Features

- Add inventory items with item name, category, purchase date, and stock status
- Manage item categories using relational database design
- View all inventory records in a structured table
- Update existing inventory information
- Delete inventory records
- Backend validation for all required fields
- MySQL JOIN operations to fetch item details with their categories
- Error handling and user-friendly responses
- Responsive and clean UI design

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- CSS3

### Backend
- Node.js
- Express.js

### Database
- MySQL

## 🗄️ Database Design

### item_types
Stores available item categories such as Electronics, Furniture, and Clothing.

### items
Stores inventory records and maintains relationships with item categories using foreign keys.
